# How to Build a Research Quality Stack

A quality stack is a repeatable pipeline, not a single checklist.

## Stack layers
1. Journal targeting layer
2. Writing and structure layer
3. Methods and statistics layer
4. Reporting standards layer
5. Integrity and citation verification layer

## Implementation order
Start with the biggest current bottleneck, then add one layer per week.

## Minimum viable stack
- journal shortlist workflow
- pre-submit writing checklist
- reporting standard mapping
- citation integrity pass
