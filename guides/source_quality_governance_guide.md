# Source Quality Governance Guide

Curation quality drops when source standards are vague.

## Governance rules
- prefer official sources
- remove dead links quickly
- avoid duplicate low-value summaries
- log update date on major changes

## Maintenance cadence
- weekly: check new additions and remove obvious weak links
- monthly: run full link and relevance audit
- quarterly: tighten inclusion criteria based on usage

## Acceptance rubric
Use `templates/curation/resource_evaluation_rubric.md` and keep only high-scoring resources.
